# Petpooja Tools

A polished static remake of the tools from `petpoojatools.blogspot.com`, built for GitHub Pages.

## Tools

- CSV / Excel splitter with custom rows per file
- 999 row correction splitter
- Column A + Column B merger into Column C
- Line-to-comma separator with copy button

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `app.js` from this folder.
3. In GitHub, open **Settings > Pages**.
4. Select the main branch and root folder, then save.

The app is browser-only. Files are processed locally in the visitor's browser.
